Utilisation de CSS

❮ Précédent Suivant ❯

Il y a trois façons d'ajouter les styler CSS au document HTML.

  • style en ligne - ajouter l'attribut style aux éléments HTML
  • style interne - ajouter l'élément <style> dans la séction <head>
  • style externe - créer un fichier extérieure de CSS

Style en ligne

Vous pouvez utiliser l'attribut style de n'importe quel élément HTML pour définir les règles de style. Ces règles seront appliquées à cet élément seulement. L'attribut style peut contenir n'importe quelle propriété CSS.

<!DOCTYPE html>
<html>
  <head>
    <title>Titre du document</title>
    <style>
      body {
        background-color: #1c87c9;
      }
      p {
        color: white;
      }
    </style>
  </head>
  <body>
    <p>Quelque information</p>
  </body>
</html>
Essayez-vous-même!

Style externe

Le style externe est largement utilisé pour appliquer les styles généraux au site web entier. Il s'applique à la création d'un fichier externe CSS qui comprend toute l'information de style. Vous pouvez créer tel type de fichier avec n'importe quel texte ou en éditeur HTML tel que "Notepad" ou "Sublime text”.

Un fichier CSS contient seulement CSS, et vous devez l'enregistrer avec l'extension .css file. Pour faire référence à la feuille de style externe à une page web vous devez utiliser la balise <link> dans la séction <head> d'un document HTML. Chaque page web doit être liée à la feuille de style.

Lors l'utilisation d'une feuille de style externe, tous les fichiers HTML sont liés à un fichier CSS en résultant dans un apparence cohérente. Si vous voulez changer le style des pages web, vous devez faire les changements correspondants dans one .css file. 

<head>
  <link rel="stylesheet" type="text/css" href="style.css">
</head>
