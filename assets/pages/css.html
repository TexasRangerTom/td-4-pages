Qu'est que CSS

Cascading style sheet

Cascading Style Sheets (CSS) est un langage de feuille de style utilisée pour décrire l'apparence et la mise en forme d'un document écrit en langue de balisage. CSS a été développé par World Wide Web Consortium (W3C). CSS définit comment les éléments HTML doivent être affichés.Les styles sont généralement enregistrés à external .css files. Les feuilles de styles externes vous donnent une opportunité de changer l'apparence et la mise en page de toutes les pages dans un site web, juste en modifiant un seul fichier.

Utilisation

mise en place

Il y a trois façons d'ajouter les styler CSS au document HTML.

  1. style en ligne - ajouter l'attribut style aux éléments HTML
  2. style interne - ajouter l'élément style dans la séction head
  3. style externe - créer un fichier extérieure de CSS

style en ligne

Vous pouvez utiliser l'attribut style de n'importe quel élément HTML pour définir les règles de style. Ces règles seront appliquées à cet élément seulement. L'attribut style peut contenir n'importe quelle propriété CSS.

style externe

Le style externe est largement utilisé pour appliquer les styles généraux au site web entier. Il s'applique à la création d'un fichier externe CSS qui comprend toute l'information de style. Vous pouvez créer tel type de fichier avec n'importe quel texte ou en éditeur HTML tel que "Notepad" ou "Sublime text”. Un fichier CSS contient seulement CSS, et vous devez l'enregistrer avec l'extension .css file. Pour faire référence à la feuille de style externe à une page web vous devez utiliser la balise dans la séction d'un document HTML. Chaque page web doit être liée à la feuille de style. Lors l'utilisation d'une feuille de style externe, tous les fichiers HTML sont liés à un fichier CSS en résultant dans un apparence cohérente. Si vous voulez changer le style des pages web, vous devez faire les changements correspondants dans one .css file.

Syntaxe

Selecteur

Le sélecteur est l'élément HTML que vous voulez styler. Cela peut être n'importe quelle balise telle que <h1>, <p>, etc. Chaque sélecteur peut avoir une ou plusieurs propriétés, souligné en rouge sur l'image.

propriété

La propriété est l'attribut de style de la balise HTML. Tous les attributs HTML sont convertis en propriétés CSS (color, border, etc.). Chaque propriété a sa valeur, souligné en bleu sur l'image

Valeur

La valeur est assigné à la propriété. Par exemple, la propriété color peut avoir soit red ou #F1F1F1 etc, souligné en vert sur l'image

Quelques selecteurs

selecteur universel

Avant de continuer avec des choses plus complexes, prenons le temps de découvrir le sélecteur universel. Il permet de cibler toutes les balises sans exception. Le caractère qui identifie le sélecteur universel est *. Il s'utilise de manière très simple.

selecteur de classe

A l'inverse d'un identifiant, un ou plusieurs éléments peuvent avoir la même classe. Le sélecteur de classe sélectionne les éléments avec un attribut de classe spécifique. Le sélecteur de classe utilise l'attribut class="" d'une balise HTML pour sélectionner une ou plusieurs balises. Pour sélectionner des éléments avec une classe spécifique, il faut utiliser le caractère point : . suivi du nom de la classe.
